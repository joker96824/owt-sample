# owt-sample

### Windows下使用说明
## 以管理员身份运行 visual studio

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### 可能需要安装yarn
```
npm install yarn
```

### 修改url
```
打开 ./src/api/rest-sample.js
将‘https://192.168.16.187:33004’修改为‘https://vps.wonderview.cn:33004’
```
